## ⭐ 기능 소개
[1]게임 시작  [2]게임 설명 [3]랭킹 확인

[1]로그인 / 회원가입 / 회원탈퇴
  * DB를 사용한 로그인/회원가입/회원탈퇴 (사용자 관리 서비스) 기능 구현
  <br>
  
[2]👀 게임 소개

  * 짧은 시간동안 노래의 전주를 듣고 제목을 맞추는 게임!
  * 게임 도중 패스와 힌트를 사용할 수 있다.
  * 단, 정답 입력시 영문은 소문자로 쓰고 띄어쓰기를 쓰지 않는다.
  * 점수는 한 문제당 10점씩 총 14문제!
